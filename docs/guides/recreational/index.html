Managers

These manager apps take advantage of the managed profile feature on Android 5.0+, which is also the base of Android for work, to create an isolated sandbox for any app with its data.

  • Clone and run multiple accounts simutaniuosly
  • Freeze or archive apps and prevent any background behaviors
  • Unfreeze apps on-demand with home screen shortcuts
  • Re-freeze marked apps with one tap
  • Hide apps

Recommended managers

  • Insular
  • Shelter

Watch partyâ€‹

Hyperbeam

Hyperbeam is a website, an SaaS platform and a tool that can be used to browse the web and watch together online. Users could enjoy movie nights online with their online friends and start a watch party.

  • Uses browsers running inside virtual machines
  • Stream synchronization is not impacted by host connection
  • Remote control is transferable to other viewers
  • Can allow up to 12 participants
